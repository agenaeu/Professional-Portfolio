Shop workload app

Extra details

`This project is a workflow control app that attempts to streamline all workflow in a shop`
This app will create a high level of effenciency within a shop environment by:
1. Assigning jobs based on the work to be done.
2. Workstations are sequentially assigned in order of work to be done.
3. Workflow is assigned for maximum effeciency and cannot be changed.
4. Job documentation is digitally attached to each job to further enhance effenciency.
The app has authentication for both employee (workflow) and admin (job planning/access):
1. There is a unified login page for both Administrator and Employees.
2. Administrators can create, assign and update jobs.
3. Employees can see work assigned to their workstation and what must be accomplished.
4. Admin login will allow you to update the entire system.
Primary Technologies Used:
- `ReactJS`
- `Node`
- `Express`

Additional Technologies Used:
- `Cloud Firestore` -is a NoSQL document database

- `Firebase Authentication` -makes building secure authentication systems easy.

- `Firebase Cloud Functions` -allow you to develop a backend without servers.

- `Firebase Cloud Storage` -allows you to store and serve content with relative ease.

- `Firebase Hosting` is fast and secure web hosting that can deliver content very quickly.

Save the trees

Extra details

In an effort to reduce waste and prevent order confusion, Team Arkie Alderton decided to create a digital sushi menu card that can be filled out through a web browser on a patron's smart phone, tablet or laptop.

Simply sit down and use a web browser to visit the URL provided at the table.

Add a quantity to a desired item and review the order in the summary bar.

When the order is submitted, the selected items, quantities and submition time are stored to a server database and then displayed on a dedicated Chef's page.

As the Chef completes an order, the items can checked off the list as complete and completion time stored to provide better time estimations and for chefs to review their performance and identify areas for improvement.
